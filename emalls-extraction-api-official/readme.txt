این پلاگین جهت دریافت تمامی محصولات فروشگاه‌های وردپرسی که از پلاگین ووکامرس استفاده می‌کنند، توسعه یافته است.

== Description ==
&#x202b;این پلاگین یک API اختصاصی برای استخراج محصولات ووکامرس ایجاد می‌کند و توسط وب‌سرویس ایمالز برای دریافت اطلاعات محصولات استفاده می‌شود.  
&#x202b;هدف این پلاگین فراهم‌کردن دسترسی سریع، امن و بهینه به اطلاعات محصولات شامل نام، قیمت، تصاویر، ویژگی‌ها، گارانتی و وضعیت موجودی است.

### قابلیت‌های نسخه 1.2.0
* بهینه‌سازی کامل ساختار API برای افزایش سرعت و پایداری  
* بهبود چشمگیر سرعت با پرایم‌کردن کش تصاویر (`_prime_post_caches`)  
* بازنویسی بخش دریافت محصولات با استفاده از کوئری‌های بهینه  
* استخراج دقیق‌تر و استانداردتر تصاویر (تصویر اصلی + گالری بدون تکرار)  
* افزودن تاریخ ایجاد و بروزرسانی محصول با فرمت استاندارد `DATE_ATOM`  
* افزودن اطلاعات متادیتا شامل نسخه وردپرس، PHP، ووکامرس و libsodium  
* بهبود استخراج ویژگی‌ها و وارییشن‌ها مطابق استاندارد ووکامرس  

### نیازمندی‌ها
* وردپرس نسخه ۵.۲.۰ به بالا  
* ووکامرس نسخه ۴.۰.۰ به بالا  

== Installation ==

1. &#x202b;پلاگین را داخل پوشه `/wp-content/plugins/` کپی کنید یا از مخزن وردپرس نصب نمایید.  
2. &#x202b;سپس پلاگین را از بخش افزونه‌ها فعال کنید.  

== External services ==
* وب‌سایت ایمالز (https://emalls.ir) یک موتور جستجوی کالا است که به فروشگاه‌ها کمک می‌کند محصولات خود را بهتر نمایش دهند.  
* این پلاگین یک API امن برای ارسال اطلاعات محصولات به ایمالز فراهم می‌کند.  
* اطلاعات شامل نام، تصویر، قیمت، مشخصات، گارانتی و وضعیت موجودی است.  
* برای جلوگیری از دسترسی غیرمجاز، ایمالز توکن و دامنه را ارسال می‌کند و این پلاگین اعتبار آن را از سرور ایمالز استعلام می‌گیرد.
