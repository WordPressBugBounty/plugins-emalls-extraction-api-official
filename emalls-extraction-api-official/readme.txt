== Description ==
&#x202b;این پلاگین جهت دریافت تمامی محصولات فروشگاه های وردپرسی که از پلاگین ووکامرس استفاده می کنند، توسعه یافته است.
&#x202b;هدف این پلاگین دسترسی به یک وب سرویس برای دریافت تمامی محصولات و به روز رسانی سریع مشخصات هر محصول است.
## نیازمندی ها
* وردپرس نسخه ۵.۲.۰ به بالا
* ووکامرس نسخه ۴.۰.۰ به بالا


== Installation ==

1. &#x202b;کافیست پلاگین را داخل پوشه `/wp-content/plugins/` کپی کنید و یا آن را از انباره پلاگین های وردپرس به صورت مستقیم نصب نمایید.
2. &#x202b;سپس پلاگین را از قسمت افزونه های داشبورد خودتان فعال نمایید.


== External services ==
* Emalls website (https://emalls.ir) is a product search engine that helps store websites to increase their sales. 
* Emalls needs this plugin to receive product information.
This information includes the name, image, price, specification and warranty of the products.
This plugin adds a section to your website through which Emalls will see the list and information of your products.
In order to prevent information disclosure to third parties, Emalls sends parameters (including token and domain name) and this plugin queries the validity of those parameters from the Emalls website.
